# Diligion Brand Identity Summary

## Brand Colors

### Primary Colors
- **Navy Blue (Primary)**: `#002b57`
  - Used for: Logo mark, primary text, headings
- **Terracotta/Orange (Accent)**: `#cb5b2b`
  - Used for: Logo accent element, call-to-action buttons, highlights
- **White**: `#fff`
  - Used for: Backgrounds, logo elements on dark backgrounds
- **Gray (Text)**: `#7b7b7b`
  - Used for: Secondary text, body content

## Typography

### English Fonts
- **Montserrat** (Variable Font)
  - Weights: Regular, Medium, SemiBold
  - Usage: Primary English font for headings and body text
- **Readex Pro** (Regular)
  - Usage: Alternative English font

### Arabic Fonts
- **IBM Plex Sans Arabic**
  - Weights: ExtraLight, Light, Regular, Medium, SemiBold, Bold, Thin
  - Usage: Primary Arabic font for all Arabic content
- **HT Rakik**
  - Weights: Regular, Medium, SemiBold
  - Usage: Alternative Arabic font (possibly for display/headings)

## Design Principles

### Logo Structure
- Circular mark with geometric design
- Navy blue circle as base
- Terracotta/orange accent shape
- White square element
- Clean, modern, professional aesthetic

### Brand Personality
- Professional
- Modern
- Trustworthy
- Diligent
- Visionary

## Usage Guidelines

### Color Usage
- Primary navy blue should be dominant
- Terracotta accent should be used sparingly for emphasis
- Maintain high contrast for readability
- Use white backgrounds for light mode
- Use navy backgrounds with white text for dark mode sections

### Typography Hierarchy
- Headings: Montserrat SemiBold/Medium (English), IBM Plex Sans Arabic Bold/SemiBold (Arabic)
- Body: Montserrat Regular (English), IBM Plex Sans Arabic Regular (Arabic)
- Ensure proper RTL support for Arabic content

### Logo Usage
- Maintain minimum clear space around logo
- Use appropriate logo version for context (full for headers, short for compact spaces, mark for favicon)
- Ensure sufficient contrast with background
- Do not distort or modify logo elements
